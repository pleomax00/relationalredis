Concepts
Relational Database
  • A relational database is a collection of data items organized as a set of formally described tables from which data can be accessed easily.
  • A relational database is created using the relational model.
Key/value Stores

Key-value stores allow the application to store its data in a schema-less way. The data could be stored in a datatype of a programming language or an object as one huge table of key and values.

Example:

password 8451ba8a14d79753d34cb33b51ba46b4b025eb81
